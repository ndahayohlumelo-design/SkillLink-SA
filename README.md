# SkillLink-SA
SkillLink SA is a **multi-page informational website** that showcases a digital solution to **youth unemployment in South Africa**.  
The website demonstrates both **business systems thinking** and **front-end development skills**, making it a strong portfolio piece for potential employers.

The website includes:

- **Home Page** – Introduction to the problem and overview of the solution.
- **The Problem** – Details on South Africa's unemployment statistics.
- **Causes** – Insights into why unemployment is so high.
- **Digital Solutions** – How technology can help solve unemployment.
- **My Solution** – A proposed platform called SkillLink SA to connect skilled candidates to job opportunities.
- **Contact Page** – Allows interested collaborators to submit their information directly via a form.

---

## Features

- **Responsive Design** – Works across mobile, tablet, and desktop devices.
- **Small, Professional Logo** – Positioned in the top-left like ChatGPT for clean branding.
- **Navigation** – Consistent multi-page navigation for smooth user experience.
- **Interactive Contact Form** – Allows visitors to submit their details via email (powered by Formspree).
- **Accessible and Semantic HTML** – Uses proper headings, labels, and alt text for images.

---

## Technologies Used

- **HTML5** – Structure of the website.
- **CSS3** – Styling, responsive design, and layout.
- **JavaScript** – For any interactive elements (smooth scrolling, optional enhancements).
- **Formspree** – To handle contact form submissions via email.
- **Google Fonts (Poppins)** – Clean and modern typography.
